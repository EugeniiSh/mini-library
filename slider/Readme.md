# Slider Widget

## Описание

Класс `Slider` используется для создания настраиваемого слайдера, который поддерживает автоматическое пролистывание, пагинацию и стрелки навигации.

## Установка

1. Скопируйте разметку, стили и код в свой проект.
2. В файле с кодом JavaScript выполните следующие шаги:

```javascript
// Создайте переменную для контейнера слайдера
const slider = document.querySelector('.slider-container');

// Создайте объект конфигурации слайдера
//(все параметры, кроме slider, не обязательные)
const sliderSetup = {
    slider: slider, //объект слайдера (например slider, из пункта выше)
    visibleNum: 1, //любое целое число. Настраивает количество одновременно видимых элементов слайдера. По умолчанию = 1. Доплнительно (при значении visibleNum > 1): в файле style.css, в селекторе .line-item__block изменить значение width так, что бы были видны все элементы в окне просмотра. Ширину каждого элемента нужно расчитать самостоятельно, с учётом величины gap, в селекторе .slider-line. 
    autoScroll: true, //булевый тип. Вкл-Откл автопролистывания слайдера. По умолчанию = true.
    paginations: true, //булевый тип. Вкл-Откл пагинанацию для слайдера. По умолчанию = true.
    arrows: true, //булевый тип. Вкл-Откл стрелок влево-вправо. По умолчанию = true.
    slideTransition: 1, //любое целое число. Время прокрутки слайдов, в секундах. По умолчанию = 1. Не должно быть больше времени переключения слайдов. Если время прокрутки больше чем время переключения слайдов, то время прокрутки будет 90% от времени переключения слайдов.
    autoScrollTime: 5, //любое целое число. Время переключения между слайдами в авто режиме, в секундах. По умолчанию = 5.
    paginationStatic: false, //Вкл-Откл статичную пагинанацию для слайдера. По умолчанию = false. Дополнительно (при значении paginationStatic: true и autoScroll: true): в файле style.css, в селекторе .pagination-block__filler изменить значение background-color на transparent, или убрать его совсем.
    infinity: false, //Вкл-Откл режим бесконечного слайдера. По умолчанию = false. Дополнительно: в этом режиме количество одновременно видимых слайдов должно быть не больше одного. С другими значениями слайдер будет работать не корректно.
    opacity: false, //Вкл-Откл режим переключения между слайдами с помощью изменения прозрачности. Свойство infinity должно быть = true. Дополнительно: в этом режиме количество одновременно видимых слайдов должно быть не больше одного. С другими значениями слайдер будет работать не корректно.
};

// Инициализируйте слайдер
const sliderObj = new Slider(sliderSetup);

// Обновление слайдера при изменении размера окна
window.addEventListener('resize', () => {
    sliderObj.setup(slider);
});
